# English For Kids - Qúi Bình Phạm

A professional website for English language courses for children aged 3 to grade 9.

## Setup Instructions

1. **Create the following files:**
   - index.html
   - courses.html
   - about.html
   - contact.html
   - styles.css
   - script.js

2. **Add your images:**
   - logo.png - Your English For Kids logo
   - zalo-qr.png - Your Zalo QR code (from Image 2)
   - facebook-qr.png - Your Facebook QR code
   - instagram-qr.png - Your Instagram QR code

3. **Update Google Maps:**
   - Replace the iframe src in contact.html and index.html with your actual Google Maps embed code
   - Go to Google Maps → Find your location → Share → Embed a map → Copy HTML

4. **Customization:**
   - Update contact information in all HTML files
   - Add your actual social media links
   - Replace placeholder images in gallery sections
